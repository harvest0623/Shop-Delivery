## 实体说明

### 1. 商家（Shops）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 商家ID | INT | 主键 | 唯一标识 |
| 商家名称 | VARCHAR | 非空 | 如：肯德基 |
| 商家描述 | TEXT | | 详细介绍 |
| 商家地址 | VARCHAR | | 实体地址 |
| 联系电话 | VARCHAR | | 客服电话 |
| 商家图片 | VARCHAR | | 展示图片URL |
| 评分 | DECIMAL | 默认4.5 | 1-5分 |
| 配送时间 | INT | 默认30 | 分钟 |
| 配送费 | DECIMAL | 默认3.00 | 元 |
| 起送价 | DECIMAL | 默认20.00 | 元 |
| 状态 | VARCHAR | 默认open | open/closed |

### 2. 商品分类（Categories）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 分类ID | INT | 主键 | 唯一标识 |
| 分类名称 | VARCHAR | 非空 | 如：汉堡、炸鸡 |
| 图标 | VARCHAR | | emoji图标 |
| 排序 | INT | 默认0 | 显示顺序 |

### 3. 商品（Products）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 商品ID | INT | 主键 | 唯一标识 |
| 商家ID | INT | 外键→Shops | 所属商家 |
| 分类ID | INT | 外键→Categories | 所属分类 |
| 商品名称 | VARCHAR | 非空 | 如：吮指原味鸡 |
| 商品描述 | TEXT | | 详细介绍 |
| 价格 | DECIMAL | 非空 | 售价 |
| 商品图片 | VARCHAR | | 展示图片 |
| 库存 | INT | 默认100 | 数量 |
| 是否招牌 | TINYINT | 默认0 | 0否/1是 |

### 4. 顾客（Customers）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 顾客ID | INT | 主键 | 唯一标识 |
| 用户名 | VARCHAR | 非空、唯一 | 登录账号 |
| 密码 | VARCHAR | 非空 | bcrypt加密 |
| 邮箱 | VARCHAR | | 联系邮箱 |
| 手机号 | VARCHAR | | 联系电话 |
| 昵称 | VARCHAR | | 显示名称 |
| 头像 | VARCHAR | | 头像URL |

### 5. 地址（Addresses）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 地址ID | INT | 主键 | 唯一标识 |
| 顾客ID | INT | 外键→Customers | 所属顾客 |
| 收件人姓名 | VARCHAR | | 收货人 |
| 收件人电话 | VARCHAR | | 联系电话 |
| 省份 | VARCHAR | | 省/直辖市 |
| 城市 | VARCHAR | | 市 |
| 区县 | VARCHAR | | 区/县 |
| 详细地址 | VARCHAR | | 街道门牌号 |
| 是否默认 | TINYINT | 默认0 | 0否/1是 |

### 6. 购物车（Cart）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 购物车ID | INT | 主键 | 唯一标识 |
| 顾客ID | INT | 外键→Customers | 所属顾客 |
| 商品ID | INT | 外键→Products | 所选商品 |
| 数量 | INT | 默认1 | 购买数量 |
| 添加时间 | TIMESTAMP | | 创建时间 |
| 更新时间 | TIMESTAMP | | 最后修改 |

### 7. 订单（Orders）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 订单ID | INT | 主键 | 唯一标识 |
| 顾客ID | INT | 外键→Customers | 下单顾客 |
| 商家ID | INT | 外键→Shops | 订单商家 |
| 地址ID | INT | 外键→Addresses | 配送地址 |
| 订单金额 | DECIMAL | 非空 | 商品总价 |
| 配送费 | DECIMAL | 默认0 | 运费 |
| 订单状态 | VARCHAR | 默认pending | pending/paid/shipping/completed/returned/cancelled |
| 订单编号 | VARCHAR | 唯一 | 如：ORD202605193295 |
| 备注 | TEXT | | 用户留言 |
| 退货原因 | VARCHAR | | 退货说明 |

### 8. 订单项（OrderItems）
| 属性 | 类型 | 约束 | 说明 |
|------|------|------|------|
| 订单项ID | INT | 主键 | 唯一标识 |
| 订单ID | INT | 外键→Orders | 所属订单 |
| 商品ID | INT | 外键→Products | 所购商品 |
| 数量 | INT | 默认1 | 购买数量 |
| 单价 | DECIMAL | 非空 | 下单时价格 |

## 联系说明

| 联系 | 实体1 | 实体2 | 基数 | 说明 |
|------|-------|-------|------|------|
| 拥有 | 商家 | 商品 | **1:N** | 一个商家有多个商品 |
| 属于 | 商品分类 | 商品 | **1:N** | 一个分类包含多个商品 |
| 拥有 | 顾客 | 地址 | **1:N** | 一个顾客有多个地址 |
| 拥有 | 顾客 | 购物车 | **1:N** | 一个顾客有多个购物车项 |
| 下单 | 顾客 | 订单 | **1:N** | 一个顾客下多个订单 |
| 接收 | 商家 | 订单 | **1:N** | 一个商家接收多个订单 |
| 配送至 | 地址 | 订单 | **1:N** | 一个地址对应多个订单 |
| 包含 | 订单 | 订单项 | **1:N** | 一个订单有多个订单项 |
| 对应 | 商品 | 订单项 | **1:N** | 一个商品出现在多个订单项 |

## E-R图转关系模式

```sql
-- 商家表
shops(商家ID, 商家名称, 商家描述, 商家地址, 联系电话, 商家图片, 评分, 配送时间, 配送费, 起送价, 状态)
主键：商家ID

-- 商品分类表
categories(分类ID, 分类名称, 图标, 排序)
主键：分类ID

-- 商品表
products(商品ID, 商家ID, 分类ID, 商品名称, 商品描述, 价格, 商品图片, 库存, 是否招牌)
主键：商品ID
外键：商家ID → shops(商家ID)
外键：分类ID → categories(分类ID)

-- 顾客表
customers(顾客ID, 用户名, 密码, 邮箱, 手机号, 昵称, 头像)
主键：顾客ID

-- 地址表
addresses(地址ID, 顾客ID, 收件人姓名, 收件人电话, 省份, 城市, 区县, 详细地址, 是否默认)
主键：地址ID
外键：顾客ID → customers(顾客ID)

-- 购物车表
cart(购物车ID, 顾客ID, 商品ID, 数量, 添加时间, 更新时间)
主键：购物车ID
外键：顾客ID → customers(顾客ID)
外键：商品ID → products(商品ID)

-- 订单表
orders(订单ID, 顾客ID, 商家ID, 地址ID, 订单金额, 配送费, 订单状态, 订单编号, 备注, 退货原因)
主键：订单ID
外键：顾客ID → customers(顾客ID)
外键：商家ID → shops(商家ID)
外键：地址ID → addresses(地址ID)

-- 订单项表
order_items(订单项ID, 订单ID, 商品ID, 数量, 单价)
主键：订单项ID
外键：订单ID → orders(订单ID)
外键：商品ID → products(商品ID)
```
